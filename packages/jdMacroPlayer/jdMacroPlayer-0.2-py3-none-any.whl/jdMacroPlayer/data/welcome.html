Welcome to jdMacroPlayer!

Please read this before using

This program allows you to simulate keyboard and mouse input by pressing a shortcut. It let's you also execute a command.

You can use this to automate things on your computer

Unfortunately Wayland does not allow simulating input. To solve this problem, ydotool is used, which creates a virtual input device. This requires root permission.

While jdMacroPlayer is designed with Wayland in mind, it also works on X11

To use jdMacroPlayer you must start the Daemon under Daemon>Start. Your Macros will only work while the Daemon is running.

jdMacroPlayer uses the Global Shortcuts portal. You need to use the Tools provided by your Desktop to configure the Shortcuts.

While jdMacroPlayer have been tested with KDE Plasma 6, it should also work on other desktops too.

jdMacroPlayer is currently in Beta, so expect some Problems.

If you encounter a bug or have a feature request, please open a Issue.

For more information take a look at the Documentation.
